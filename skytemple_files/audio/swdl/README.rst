SWDL File Format
================
SWDL Format of the Procyon Studios Digital Sound Elements sound driver.

Usage
-----
Use the class ``SwdlHandler`` of the ``handler`` module, to open and save
models from binary data.

File Format
-----------
See https://projectpokemon.org/home/docs/mystery-dungeon-nds/dse-swdl-format-r14/.
